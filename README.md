# 🚀 Premium Dark-Mode Fitness SaaS App & Developer Template

A fully functional, production-ready **Next.js 14+** mobile-first web template integrated with a serverless **Supabase** backend, user authentication, and global state tracking.

## 📱 Features
* **Dynamic Home Dashboard:** Real-time macro tracking, functional calorie balance indicators, and responsive inline custom food-logging forms.
* **Hydration Tracker:** Visual fluid-intake panel featuring single-click data increments and reset triggers.
* **Custom Training Module:** An exercise creator that calculates and logs active energy expenditure instantly.
* **YouTube Video Library Reference:** A built-in video feed section allowing users to store and stream curated video workouts directly within the app shell.
* **Live Analytics Panel:** Modern UI charts and progress bars that render historic stats on-the-fly using live application data.
* **Secure Profile Syncing:** Fully integrated cloud user registration, instant sign-in, and dynamic profile data updates.

## 🛠️ Tech Stack
* **Frontend:** Next.js (App Router, React, TypeScript)
* **Styling:** Tailwind CSS (Modern Dark-Mode Aesthetic)
* **Backend & Database:** Supabase (PostgreSQL, Auth, Serverless Real-time API)
* **State Management:** React Context API + LocalStorage fallback
